# Summary

This is a Dockerised expressjs application

## Dependencies

Docker desktop

## How to run

* `docker build -t react-express-node .`
* `docker run -it -p 5000:8080 react-express-node`
* Go to localhost:5000 in your browser
* You should see a welcome message
